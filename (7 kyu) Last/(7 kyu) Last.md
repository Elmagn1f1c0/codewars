# Last (7 kyu)

https://www.codewars.com/kata/541629460b198da04e000bb9

Find the last element of the given argument(s).

Examples:

```js
last([1, 2, 3, 4]); // =>  4
last('xyz'); // => "z"
last(1, 2, 3, 4); // =>  4
```

In `javascript` and `CoffeeScript` a list will be an array, a string or the list of arguments.
